

### 💻 프로젝트 소개

+ jsp를 활용해 로그인과 회원가입, 게시글 CRUD, 관리자 권한으로 회원관리 할수있는 사이트를 만들었습니다.
+ 국민신문고 민원사이트로 불편한 점이 있을때 민원을 신청하고 공공문제에 대해 다른 사람들과 이야기를 나누고 싶을 때 생각을 게시함으로 민원이나 제안을 국민신문고에 제출하여 처리하도록 합니다.
